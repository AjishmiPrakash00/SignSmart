# SheSign project requirements (full-stack)
# This file lists the requirements and installation steps for the entire project (backend + frontend).

# ----- Backend (Python) -----
# Install these in a Python virtual environment:
# Example:
# python -m venv .venv
# source .venv/bin/activate   # (macOS / Linux)
# .venv\Scripts\activate    # (Windows PowerShell)
# pip install -r backend/requirements.txt

# The backend requirements are available in: backend/requirements.txt
# (If you prefer a single pip install, uncomment and run the following pip line in the backend venv)
# fastapi
# uvicorn[standard]
# python-multipart
# pdfplumber
# pytesseract
# Pillow

# NOTE: pytesseract requires the Tesseract OCR engine installed on your system.

# ----- Frontend (Node) -----
# The frontend uses Node packages managed by npm. Run these commands from the `frontend` folder:
# cd frontend
# npm install
# npm run dev
